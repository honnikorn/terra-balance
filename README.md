# 🌍 Terra Balance

**Educational climate strategy game** — build a civilisation, protect the planet, survive to the end.

**[▶ Play Now](https://honnikorn.github.io/terra-balance/)**

## Features
- 3D interactive globe (Three.js)
- 37 buildings · 20 technologies in 5 tiers
- 20 real climate policies
- 51 crisis events with real-world science
- 40 quiz questions with learning reports
- 16 achievements
- 20-entry climate encyclopedia
- Hot-seat 2-player multiplayer
- Climate trend charts
- Sound effects (Web Audio — no downloads)
- First-play tutorial
- Year-end summaries
- Works offline after first load

## Updating
After a new build from Claude:
```bash
./update.sh ~/Downloads/terra_balance_globe.html
```
Or just drag `index.html` into the GitHub web interface.

## Setup (one time)
1. Go to **Settings → Pages** in this repo
2. Set Source to **GitHub Actions**
3. Done — the workflow auto-deploys on every push
